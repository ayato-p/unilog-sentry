# unilog-sentry

Sentry appender for pyr/unilog (wip)

## Installation

unilog-sentry is [available on Clojars](https://clojars.org/cc.qbits/unilog-sentry).

Add this to your dependencies:

```clojure
[cc.qbits/unilog-sentry "0.1.0-SNAPSHOT"]
```
